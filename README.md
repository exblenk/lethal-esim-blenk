# ExBlenk

Activate eSIMs instantly worldwide. No physical SIMs, no roaming fees.

## Features
- Instant eSIM activation  
- Global data plans  
- Secure checkout  
- User dashboard  

## Tech Stack
- React + TypeScript  
- Tailwind CSS  
- Vite  
- shadcn-ui  

## Run Locally
```bash
git clone https://github.com/exblenk/lethal-esim-blenk.git
cd lethal-esim-blenk
npm i
npm run dev
